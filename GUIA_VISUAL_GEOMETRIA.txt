🎯 CONSEJOS PRACTICOS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ANTES DE HACER CLICKS:

1. USA ZOOM para precision
   Rueda del raton: Acercar/Alejar
   +/- en teclado: Pasos fijos

2. DESPLAZA la imagen si es necesario
   Flechas: Pan en pasos
   Boton derecho + arrastrar: Pan libre

3. VERIFICA que estas en el modo correcto
   Panel superior muestra: "MODO ACTIVO: GEOMETRÍA - CIRCULO"


DURANTE LOS CLICKS:

✓ Haz click EXACTAMENTE en el punto deseado
✓ Los puntos aparecen marcados en la imagen
✓ Para poligono, veras las lineas conectandose

SI TE EQUIVOCAS:

✗ Presiona 'r' para limpiar y empezar de nuevo
✗ O simplemente presiona el numero de figura nuevamente (1-4)


VISUALIZACION:

Los puntos temporales (mientras mides) aparecen en:
  - CYAN para modo geometria
  - Con numeros (1, 2, 3, ...)

El resultado final aparece en:
  - CYAN para circulos
  - AMARILLO para rectangulos
  - NARANJA para triangulos y poligonos

📐 FORMULAS MATEMATICAS USADAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CIRCULO (desde 3 puntos):

Dados 3 puntos (x1,y1), (x2,y2), (x3,y3) en el perimetro:

  d = 2 * (x1*(y2-y3) + x2*(y3-y1) + x3*(y1-y2))

  Centro:
    ux = ((x1²+y1²)*(y2-y3) + (x2²+y2²)*(y3-y1) + (x3²+y3²)*(y1-y2)) / d
    uy = ((x1²+y1²)*(x3-x2) + (x2²+y2²)*(x1-x3) + (x3²+y3²)*(x2-x1)) / d

  Radio = distancia((x1,y1), (ux,uy))
  Diametro = 2 * Radio
  Area = π * Radio²


TRIANGULO (Formula de Heron):

Dados los 3 lados a, b, c:

  s = (a + b + c) / 2  ← Semi-perimetro

  Area = √(s * (s-a) * (s-b) * (s-c))


POLIGONO (Formula de Shoelace):

Dados N vertices (x0,y0), (x1,y1), ..., (xN-1,yN-1):

  Area = 0.5 * |Σ(xi * yi+1 - xi+1 * yi)| para i=0 hasta N-1

  donde xN = x0 y yN = y0 (cierra el poligono)
