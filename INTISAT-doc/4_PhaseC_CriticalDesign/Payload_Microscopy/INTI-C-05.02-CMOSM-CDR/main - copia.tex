\section*{Resumen}
Este documento presenta la \textbf{Revisión Crítica de Diseño (CDR)} del 
subsystema \textbf{CMOS Microscopy Payload} perteneciente al CubeSat INTISAT.
Se describe la arquitectura óptica, electrónica, mecánica y de software, así 
como los requerimientos, verificaciones, riesgos y pruebas de validación.
El contenido se estructura siguiendo lineamientos NASA/ECSS y el formato 
DDJF–PL del proyecto AcubeSAT.

\vspace{0.3cm}
\noindent\textbf{Palabras clave:} Fourier Ptychography, CMOS sensor, OLED illumination,
super-resolución, microscopía computacional.
