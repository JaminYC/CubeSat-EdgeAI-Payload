\section*{Resumen}
Este documento presenta la \textbf{Revisión Crítica de Diseño (CDR)} del
\textbf{CMOS Microscopy Payload (CMOSM)} del CubeSat INTISAT. El objetivo del
payload es demostrar un sistema de \textbf{microscopía sin lentes (lensless) en
configuración de contacto}, basado en un sensor CMOS comercial, capaz de capturar
estructuras celulares de muestras biológicas planas.

El diseño prioriza la reproducibilidad experimental mediante control de
iluminación, registro de parámetros de captura y el adecuado \textbf{acoplamiento
óptico muestra--sensor}, identificado como un factor crítico para maximizar el
contraste y la definición de bordes. Sobre la imagen adquirida, se evalúan
técnicas de realce computacional, incluyendo \textbf{super--resolución basada en
aprendizaje profundo}, como etapa complementaria para mejorar la legibilidad
visual de estructuras ya presentes en la captura original. Técnicas de
reconstrucción avanzadas como FPM se consideran dentro de un \textbf{modo
experimental} y no constituyen el mecanismo primario de formación de imagen en
la fase actual.

\vspace{0.3cm}
\noindent\textbf{Palabras clave:} microscopía sin lentes, microscopía de contacto, sensor CMOS, acoplamiento óptico, iluminación estructurada, super--resolución.
